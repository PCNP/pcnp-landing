About

Данная система помогает упростить процессы оформления документов для ООО и ИП.

    Основные модули системы:
  • 1) Модуль ООО - позволяет собрать документы в автоматическом режими для взаимодействия с ООО
  • 2) Модуль ИП - позволяет собрать документы в автоматическом режими для взаимодействия с ИП
  • 3) Модуль стоимости. Информационный модуль с тарифами
  • 4) Модуль проверки контрагента
  • 5) Модуль о нас. Предосталяет основную информацию о компании.
